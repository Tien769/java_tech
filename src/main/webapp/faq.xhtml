To ensure your package is returned correctly please include the return slip included at the bottom of your paper receipt. Return the package to the address indicated below and include and display the order number prominently on the slip and the packaging. Return postage is to be covered by the sender. If you have any further questions regarding your return please reach out to: rusticcorner@gmail.com.
